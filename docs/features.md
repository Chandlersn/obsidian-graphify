# 功能详解

## 核心功能一览

| 功能     | 快捷键        | 说明                 |
| ------ | ---------- | ------------------ |
| 搜索节点   | `Ctrl+K`   | 快速搜索，实时筛选          |
| 篮选分类   | 点击按钮       | 按分类显示/隐藏节点         |
| 查看详情   | 单击节点       | 弹窗显示笔记预览           |
| 打开笔记   | 双击节点       | 跳转 Obsidian 编辑     |
| 路径追踪   | Sidebar 按钮 | 计算两点间最短 Wiki 链接    |
| 热度统计   | Sidebar 按钮 | 显示点击次数热力图          |
| 影响力分析  | Sidebar 按钮 | PageRank 排名 Top 10 |
| 孤岛检测   | Sidebar 按钮 | 高亮无连接节点            |
| 导出 PNG | Sidebar 按钮 | 保存图谱图片             |
| 主题切换   | Sidebar 按钮 | 深色/浅色模式            |

***

## 详细说明

### 1. 搜索节点 `Ctrl+K`

按下 `Ctrl+K`，顶部搜索框激活。

输入关键词，图谱实时筛选：

- 匹配节点名称
- 高亮匹配结果
- 非匹配节点隐藏

**使用场景**：快速定位某个知识点

***

### 2. 筛选分类

点击分类按钮，只显示对应分类的节点：

```
[全部] [系统] [AI/ML] [架构] [工具] [Graphify] ...
```

**动态渲染**：按钮从实际数据生成，新增笔记自动出现新分类。

**使用场景**：聚焦某个领域，比如只看 AI 相关笔记

***

### 3. 查看详情（单击节点）

单击任意节点，弹窗显示：

- 笔记标题
- 所属分类
- 预览内容（前 100 字）
- Wiki 链接列表

**使用场景**：快速预览笔记内容，不离开图谱

***

### 4. 打开笔记（双击节点）

双击节点，通过 `obsidian://` URL scheme 打开 Obsidian：

```
obsidian://open?vault=yourvault&file=path/to/note
```

**使用场景**：从图谱直接进入编辑

***

### 5. 路径追踪

点击 Sidebar 的"路径"按钮：

1. 弹窗显示操作说明
2. 点击第一个节点作为起点
3. 点击第二个节点作为终点
4. 显示最短 Wiki 链接路径

**算法**：BFS（广度优先搜索）

**使用场景**：发现知识脉络，比如"LLM → Agent → 飞书集成"

***

### 6. 热度统计

点击 Sidebar 的"热度"按钮：

- 显示每个节点的点击次数
- 颜色深浅表示热度高低
- 数据存储在 localStorage

**使用场景**：知道你最常访问哪些知识点

***

### 7. 影响力分析

点击 Sidebar 的"影响力"按钮：

- PageRank 式影响力计算
- 连接数越多，影响力越高
- 显示 Top 10 排名

**使用场景**：发现核心知识点

***

### 8. 孤岛检测

点击 Sidebar 的"孤岛"按钮：

- 高亮无连接的节点（红色边框）
- 发现需要建立链接的内容

**使用场景**：找到"孤立"的笔记，建立连接

***

### 9. 导出 PNG

点击 Sidebar 的"导出"按钮：

- 使用 canvas.toDataURL()
- 保存为 PNG 图片
- 可分享知识结构

**使用场景**：分享图谱给别人看

***

### 10. 主题切换

点击 Sidebar 的"主题"按钮：

- 深色模式：`#08090a` 背景
- 浅色模式：`#ffffff` 背景
- 所有元素同步切换

**使用场景**：适配不同环境，保护眼睛

***

## 关系线类型

图谱中的连线代表不同关系：

| 类型         | 颜色 | 说明              |
| ---------- | -- | --------------- |
| wiki-link  | 灰色 | `[[Wiki链接]]` 连接 |
| reference  | 蓝色 | 参考引用关系          |
| dependency | 绿色 | 依赖关系            |
| related    | 橙色 | 相关主题            |
| extends    | 紫色 | 扩展概念            |
| implements | 红色 | 实现关系            |

***

## 键盘快捷键

| 快捷键      | 功能      |
| -------- | ------- |
| `Ctrl+K` | 搜索节点    |
| `ESC`    | 关闭弹窗/搜索 |
| `Ctrl+R` | 重置图谱    |

***

## 状态提示

左上角显示当前状态：

```
● 图谱加载成功
● 筛选完成：15 个节点
● 路径已显示
```

颜色指示：

- 绿色：成功
- 橙色：警告
- 红色：错误
